LinkedDataHub is open source software you can use to manage data,
create visualizations and build apps on RDF Knowledge Graphs

Get started  
For researchers and data scientists

RDF-native notebook environment

  • Copy Linked Data resources into your personal dataspace
  • Create data-driven content interactively
  • Create customized collections backed by SPARQL queries
  • Navigate SPARQL results intuitively using the parallax navigation
  • Import tabular data and turn it into Linked Data
For developers and data publishers

Low code application development

  • Customize the layout with few lines of declarative stylesheets
  • Configure data editing forms using SPARQL-based templates
  • Create or import ontologies to model your domain
  • Use the standard SPARQL 1.1 APIs to query and manage data
  • Control access using built-in authentication and authorization

Powerful features

User interface
  • Form-based RDF editing
  • Text search and resource lookup
  • HTML and SPARQL editors
  • SPARQL-based interactive charts
  • Parallax navigation
  • Faceted filters for SPARQL results
Data management
  • Block-based structured content editor
  • Integrated Linked Data browser
  • RDF resource copy into the local dataspace
  • Ontology editor
  • SPIN and SHACL data quality constraints
  • Imports of CSV and RDF data
Development
  • WebID and OpenID Connect authentication
  • Graph-level access control and requests
  • Rich library of of extensible UI components
  • Ontology-driven domain model
  • SPARQL-based templates for editing forms
  • Command line interface
Infrastructure
  • SPARQL endpoint
  • Read-write Linked Data API
  • Linked Data proxy
  • SPARQL result cache
  • Docker-based deployment
  • AWS-based deployment
